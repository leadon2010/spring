# AI 융합 응용소프트웨어개발자 양성과정 (NCS 기반 예시 커리큘럼)

## 1. 과정 개요

* **과정명**: AI 융합 응용소프트웨어개발자 양성과정
* **훈련목표**

  * 기존 응용소프트웨어 개발 역량(NCS)을 기반으로
  * AI(API·자동화·보조개발 도구)를 실무에 적용할 수 있는 개발자 양성
* **훈련대상**

  * 비전공자 및 전공자 혼합 가능
  * 웹/프로그래밍 기초 수준 이상 권장
* **총 훈련시간**: 800~1000시간 (예: 6개월)
* **훈련유형**: 국가기간·전략산업직종 / K-Digital 가능

---

## 2. NCS 직무 연계 구조

* **주 NCS 직무**: 응용소프트웨어개발 (20010202)
* **보조 NCS**:

  * 데이터베이스 구현
  * UI/UX 구현
  * 정보시스템 운영
  * AI 활용 직무능력(융합)

---

## 3. 전체 커리큘럼 로드맵

| 단계  | 구분       | 핵심 키워드        |
| --- | -------- | ------------- |
| 1단계 | 개발 기초    | 프로그래밍, 웹, DB  |
| 2단계 | 응용SW 실무  | 백엔드, 프레임워크    |
| 3단계 | AI 기초 활용 | GPT, API, 자동화 |
| 4단계 | AI 융합 개발 | 기존 시스템 + AI   |
| 5단계 | 프로젝트     | 실무형 결과물       |

---

## 4. 단계별 상세 교과 구성

### ▶ 1단계. 응용소프트웨어 개발 기초 (200시간)

**교과목 1. 프로그래밍 기초**

* Python 문법 기초
* 제어문 / 함수 / 자료구조
* 파일 입출력
* 객체지향 개념

**교과목 2. 웹 개발 기초**

* HTML5 / CSS3
* JavaScript 기초
* DOM 제어

**교과목 3. 개발 환경 이해**

* Git / GitHub
* IDE 사용법
* 개발자 협업 개념

---

### ▶ 2단계. 응용소프트웨어 실무 개발 (250시간)

**교과목 4. 백엔드 개발**

* Python 기반 백엔드 (Flask 또는 FastAPI)
* REST API 설계
* 요청/응답 처리

**교과목 5. 데이터베이스 구현**

* RDBMS (MySQL/PostgreSQL)
* SQL 작성
* ERD 설계
* 트랜잭션 개념

**교과목 6. 응용SW 구현 실습**

* 로그인/권한 처리
* CRUD 서비스 구현
* 예외 처리

---

### ▶ 3단계. AI 기초 활용 (150시간)

**교과목 7. AI 이해 및 활용 기초**

* AI 개념 정리 (ML/DL/LLM)
* 생성형 AI의 한계와 특징
* AI 활용 시 보안·저작권

**교과목 8. GPT API 활용**

* OpenAI API 구조 이해
* 프롬프트 설계 기초
* 응답 파싱 및 후처리

**교과목 9. AI 기반 개발 보조**

* AI 코드 보조 활용
* 테스트 코드 생성
* 문서 자동화

---

### ▶ 4단계. AI 융합 응용소프트웨어 개발 (200시간)

**교과목 10. 기존 시스템에 AI 기능 결합**

* 검색 기능 고도화 (AI 검색)
* 사용자 질문 응답 기능
* 문서 요약 서비스

**교과목 11. 업무 자동화 서비스**

* 엑셀/CSV 처리 자동화
* 보고서 생성 자동화
* 사내 업무 챗봇

**교과목 12. AI 적용 설계 실습**

* AI 적용 가능 업무 도출
* 요구사항 정의
* 성능/비용 고려 설계

---

### ▶ 5단계. 실무 프로젝트 (200시간)

**프로젝트 주제 예시**

* AI 고객 상담 웹 서비스
* 내부 문서 검색·요약 시스템
* AI 기반 학습 관리 시스템

**프로젝트 산출물**

* 요구사항 정의서
* DB 설계서
* API 명세서
* 소스코드
* 시연 영상

---

## 5. 평가 방법 (NCS 기준)

* 수행평가 70%
* 프로젝트 결과물 20%
* 태도·협업 10%

---

## 6. 수료 후 진출 분야

* 응용소프트웨어 개발자
* AI 활용 개발자 (주니어)
* 업무 자동화 개발자
* AI 융합 시스템 운영자

---

## 7. 본 과정의 차별성

* ❌ AI 모델 개발 중심 아님
* ⭕ **기존 응용SW + AI 실무 활용 중심**
* ⭕ 훈련기관 운영·심사에 유리한 구조
* ⭕ 비전공자 취업 연계 현실적

---

## 8. 강사 포지션 정리

> “AI를 직접 만드는 개발자”가 아니라
> **“AI를 실무 시스템에 안전하게 붙일 수 있는 응용소프트웨어개발자”를 양성**
